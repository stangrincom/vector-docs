# Certificates

# Get certificates

  • URL: /v1/certificates
  • Method: GET
  • Headers:
    • Authorization: Bearer [YOUR_ACCESS_TOKEN] required
    • Content-Type: application/json

# Response

  • Status: 200 OK
  • Type: json

# Response body

Field Type Description
id uuid -
vendor string Name vendor certificate
unit enum nominal unit (ruble, month)
value decimal nominal value
price decimal -
